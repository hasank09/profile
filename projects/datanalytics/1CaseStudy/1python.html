Case Study¶

Design marketing strategies aimed at converting casual riders into annual members¶

Company : Divvy¶

About Data¶

Divvy's trip data is available for public use for Analysis Choose any year or Q
Note: Year 2016 is considered for this analysis after analysis these step can be applied to wide range data

Data Downloaded¶

Data has more than one file and divided in Quater
trips file >> all trips taken records
station file >> station details

Files are stored in working directory (in project directory anywhere)

To make analysis scalable the following methodology adopted as combined data is too large will take time in every step, so we took subset of data when analysis is done for Q1. We may add remaining files or any number of files in the working directory.

Analysis Approach¶

As we want to convert causal riders into Members for that we have to analyze members data and look for main preference of the customer we can find from members data and advice marketing strategy for causal riders
